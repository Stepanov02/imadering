Программа IMadering - бесплатный мультипротокольный клиент c открытым исходным кодом (open source) для общения в сети интернет. Поддерживает протоколы: ICQ (OSCAR), Jabber (XMPP). В будущем будет добавлена поддержка протокола: Mail.ru Agent(MRA).


IMadering - маленький, не требовательный к системным ресурсам, клиент идеально подходящий для использования на нетбуках.

О программе ...

Основная идея написания этой программы следующая:
Создать клиент для всех популярных протоколов общения в сети интернет. Чтобы этот клиент максимально экономил интернет трафик и ресурсы системы. Чтобы он быстро работал. Чтобы в нем были реализованы различные дополнительные возможности протоколов. Также отсутствие рекламы. Вот основные задачи.
Конечно, есть конкурентные разработки, такие как: QIP, Miranda, R&Q. Но у них нет всех вышеперечисленных качеств вкупе. Постоянно чего-то не хватает, а реализовать в них это не возможно по разным причинам. Например, из-за закрытости исходного кода (как с QIP). Поэтому было решено писать всё самому, чтобы иметь полную власть над программой и уже самому убедиться в том, что можно реализовать, а что нет.

Реализованные возможности программы:
  • Поддержка ICQ протокола последней (расширенной) редакции.
  • Поддержка Jabber протокола.
  • Отображение до 200 временных контактов из серверного списка учётной записи ICQ (помогает восстановить случайно утраченные контакты).
  • Хранение истории сообщений в сжатом формате (уменьшает размер файлов с историей сообщений в 25 раз).
  • Поддержка прокси.
  • Подсчёт трафика.
  • Окно чата с закладками и поддержкой анимированных смайликов.
  • Отправка произвольных пакетов данных для отладки и исследования ICQ протокола.
  • Статус "Нестабильный" (автоматическая смена вашего статуса для всех на "В сети" - "Не в сети" через заданный интервал времени без переподключения к серверу).
  • Программа работает быстрее своих конкурентов и потребляет меньше системных ресурсов.
  • Поддержка смены тем иконок в программе и различных наборов смайликов.
  • Установка расширенной информации о себе с сохранением на ICQ сервер (расширения новой версии протокола ICQ: количество детей, религия, курение, рост и т.д.).
  • Возможность отправки SMS сообщений (при указанном у контакта номере телефона).
  • Возможность удалить себя из чужого списка контактов.
  • Поддержка различных языков для интерфейса программы.
Это далеко не полный список возможностей программы которые постоянно дополняются новыми.
